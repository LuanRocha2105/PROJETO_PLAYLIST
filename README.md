# PROJETO_PLAYLIST

Projeto de Estrutura de Dados — FATEC Rio Claro.

Backend de um aplicativo de músicas que permite ao usuário gerenciar uma biblioteca de faixas, montar filas de reprodução por humor (baseadas no BPM) e consultar o histórico.

## Estruturas utilizadas
- Lista encadeada simples para a biblioteca de músicas.
- Fila FIFO (implementação própria) para as filas de humor e para o histórico.

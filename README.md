# PROJETO_PLAYLIST

Projeto da disciplina **Estrutura de Dados** — FATEC Rio Claro.

Aplicativo de músicas que permite gerenciar uma biblioteca de faixas, montar filas de reprodução por humor (baseadas no BPM) e consultar o histórico.

## Estruturas implementadas
- **Lista encadeada simples** para a biblioteca de músicas (`Biblioteca` + `NodoLista`).
- **Fila FIFO** própria (`Fila` + `NodoFila`) usada para as 4 filas de humor e para o histórico.
